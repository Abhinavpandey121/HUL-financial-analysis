# HUL Financial Analysis — FY2023 to FY2025

## Project Overview
A comprehensive financial analysis of Hindustan Unilever Limited (HUL)
covering three fiscal years (FY2023–FY2025).

## Contents
- HUL_Financial_Analysis.xlsx — 3-year financial statements + ratio analysis
- HUL_Analysis_Paper.pdf — Written analysis paper

## Analysis Includes
- Income Statement, Balance Sheet, Cash Flow Statement
- Liquidity, Solvency, Profitability, Efficiency & Market Value Ratios
- DuPont Analysis (ROE decomposition)
- SWOT Analysis
- Competitor Benchmarking
- Future Outlook & Recommendations

## Tools Used
- Microsoft Excel (financial modelling & ratio calculation)
- Data Sources: HUL Annual Reports FY2023–FY2025, BSE India
